# @higgsfield/quanta

Higgsfield design tokens + theme runtime. Source of truth for color,
typography, spacing, breakpoints, z-index, and border-width.
Framework-agnostic — works on top of Tailwind v4 or plain CSS.

## Quick start

```tsx
<div className="bg-q-background-primary text-q-text-primary p-4 rounded-lg">
  <h2 className="text-q-headline-md-semi-bold">Hello quanta</h2>
  <p className="text-q-body-md-regular text-q-text-secondary">World</p>
</div>
```

> **The `q-` namespace:** design-semantic utilities (color, typography, z-index,
> border-width, components) are prefixed with `q-` so they coexist with the
> legacy `@higgsfield/ui` system in one Tailwind build. Structural primitives
> (spacing, breakpoints) stay native, no prefix. Details: [`ai/AGENTS.md`](./ai/AGENTS.md).

## Requirements

- Tailwind **4.1+** (uses `@theme`, `@utility`, `@source`)
- An HTML host for the bootstrap script — any framework or vanilla

## Browse tokens

```bash
yarn workspace @higgsfield/quanta storybook
```

Opens Storybook at `http://localhost:6007` with stories for every category
(color / typography / spacing / border-width / z-index / breakpoint /
nested-themes / runtime-theme). Toggle the theme from the toolbar.

Alternatively, read the source directly: `src/css/tailwind/*.css`.

## Token taxonomy

Design-semantic categories below are `q-`-prefixed in their utilities
(`bg-q-*`, `text-q-*`, `z-q-*`, `border-q-*`). Spacing and breakpoints are native.

### Color — `bg-q-*`, `text-q-*`, `border-q-*`, `ring-q-*`, `outline-q-*`

```
--color-q-background-{primary,secondary,secondary-strong,tertiary,inverse,glass,elevated-start,elevated-end}
--color-q-text-{primary,secondary,tertiary,inverse,disabled,link,brand,danger,success,warning,info,on-overlay-secondary,on-overlay-tertiary}
--color-q-border-{default,subtle,strong,focus,error,warning,success,inverse}
--color-q-state-{error,success,warning,info}-{bg,fg,fg-soft,glow}
--color-q-brand-{primary,blue,cyan,green,orange,pink,red,yellow,yellow-light}
--color-q-{blue,red,green,grey,cool,...}-{100..1200}    ← raw palette, use sparingly
```

### Typography — `text-q-{role}-{size}-{weight}`

Composite utilities that bundle font-size, line-height, letter-spacing, weight,
and family in one class:

```
Role:    display | headline | title | body | label | caption | mono | accent
Size:    lg | md | sm   (caption: sm only)
Weight:  black | bold | semi-bold | medium | regular
```

Examples: `text-q-display-lg-bold`, `text-q-headline-md-semi-bold`,
`text-q-body-sm-regular`, `text-q-caption-sm-medium`.

Always use the **composite** rather than separate size + weight — the composite
also sets line-height, letter-spacing, and family.

### Spacing — native (no `q-`)

Named steps via standard Tailwind syntax: `p-{0,0_5,1,1_5,2,2_5,3,4,5,6,7,8,10,12,14,16,20,24}`,
plus `m-*`, `gap-*`, `w-*`, `h-*`, `inset-*`. `_` ≡ `.`, so `p-1_5` and `p-1.5`
are equivalent. Off-scale (`p-9`, `p-11`) doesn't compile — `--spacing: initial`
resets the dynamic default.

### Breakpoint — native (no `q-`)

3 responsive variants: `tablet:` (≥768px), `desktop:` (≥1280px), `wide:` (≥1920px).
`mobile:` is intentionally not emitted — mobile is the implicit base in
mobile-first design.

### Z-index — `z-q-*`

8 semantic keys (values 0..70 in steps of 10):

```
z-q-{base,dropdown,sticky,overlay,modal,popover,toast,tooltip}
```

Coexists with Tailwind's default `z-N`. Semantic names are the canonical choice.

### Border-width — `border-q-*`

3 utility classes + per-side variants:

```
border-q-{thin,medium,thick}                  → 1px, 1.5px, 2px
border-q-{l,r,t,b}-{thin,medium,thick}        ← per-side, auto-generated by Tailwind
```

For zero width use Tailwind's native `border-0`.

## Package exports

| Path | Contents |
|---|---|
| `@higgsfield/quanta/tailwind.css` | `@theme` + `@utility` for Tailwind v4 |
| `@higgsfield/quanta/theme.css` | semantic aliases (color/typography by `data-theme`) |
| `@higgsfield/quanta/primitives.css` | raw literal values (`--hf-color-grey-900` etc.) |
| `@higgsfield/quanta/runtime` | `ThemeController`, `bootstrapScript`, `defineTheme`, `readInitialThemeState` |
| `@higgsfield/quanta/<component>` | React components — e.g. `/button`, `/dropdown` |

## Runtime — quick reference

Anti-FOUC bootstrap — a generated string, inject into the `<head>` of any framework:

```ts
import { bootstrapScript } from '@higgsfield/quanta/runtime'

const inlineScript = bootstrapScript() // → plain JS string
// Inject in a <script> tag in <head> synchronously, before first paint
```

Programmatic theme control:

```ts
import { ThemeController } from '@higgsfield/quanta/runtime'

const controller = new ThemeController({ brand: 'default' })
controller.setPref('dark')                       // 'auto' | 'light' | 'dark'
controller.subscribe(state => console.log(state.resolvedTheme))
```

Dynamic theme (AI-generated / preview):

```ts
import { defineTheme } from '@higgsfield/quanta/runtime'

defineTheme('ai-ocean', {
  'background-primary': '#001a2a',
  'text-primary': '#e0f4ff',
})
controller.setOverride('ai-ocean')
```

Full API reference — JSDoc in [`src/runtime/`](./src/runtime/).
